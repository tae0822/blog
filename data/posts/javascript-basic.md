## What is JavaScript?

JavaScript (JS) is a versatile, lightweight programming language that can run in web browsers as well as other environments like Node.js, Apache CouchDB, or Adobe Acrobat. It’s a single-threaded, dynamic language that supports multiple programming styles, including object-oriented, imperative, and functional approaches.  

JS allows you to write interactive features on websites—like animations, clickable buttons, and pop-up menus—but it can also handle server-side tasks with Node.js or other platforms.  

> Note: This section focuses on the JavaScript language itself, not the browser-specific APIs. For browser-related functionality, check out [Web APIs and DOM](https://developer.mozilla.org/en-US/docs/Web/API).

The official standards for JavaScript are defined in **ECMAScript (ECMA-262)** and the **Internationalization API (ECMA-402)**. MDN documents features as soon as they’re implemented in browsers, so you might see examples using newer JavaScript features even before the specifications are finalized.  

⚠️ Remember: JavaScript is completely different from Java. Although both names are trademarks of Oracle, their syntax, semantics, and use cases are very different.
