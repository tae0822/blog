## 🔥 3.Engineering Challenges

**Optimizing Initial Load & State Synchronization**
Initially, I encountered a minor "flickering" issue where the UI would briefly show an empty state before loading data from LocalStorage. 

**Solution**: Used React’s lazy state initialization (`useState(() => ...)`) to read from LocalStorage during the initial render phase. This eliminated UI flickering and ensured consistent state hydration from the very first paint.
